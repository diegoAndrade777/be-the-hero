## Instalation
To install the dependencies and to execute the **Backend** (developer mode), clone the project on your computer an execute:
```bash
cd backend
npm install
npm start
```
To run the React's **Frontend** use:
```bash
cd frontend
npm install
npm start
```
When the process end, automatically will open the page `localhost:3000` on your web browser with de Project Be The Hero.
